シフト作成アプリ仕様書

今回のアプリはシフトを作成するアプリで、要件としては以下のものを想定。
1.日ごとに従業員を割り当てる
※今回のアプリ作成に関して、依頼者の会社の状況に合わせたものを最初は作成する
日ごとに割り当てる社員数は６人（朝・昼・夜に各２人ずつ）

2.日ごとにシフトがあり、そこに従業員を割り当てる
※一日に必要な人数が今回は６人なので、６人になるように現在はアプリを作成している
→最終的には一日に必要な人数はユーザーが指定をしてそれ通りになるようにする

3.新人社員には上司がつくものとして、シフトを組む際に条件がある
※朝・昼勤務のみ→理想はシフトの条件を任意に設定をして、組めるようにする

4.基本は３日出勤をして、２日休みとする
※現在はこの型にはめてシフトを作成しているが、最終的にはユーザーの設定人数で行えるようにするものとする

5.画面はスマートデバイスの利用を想定しているので、作成する

6.アプリはスマートデバイスおよびパソコン上で使用できるようにする
